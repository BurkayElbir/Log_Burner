# Log_Burner

This is a tool for cleaning log files including:
-general log files
-Samba logs
-PostgreSQL logs
-binary logs
-systemd journal
-bash history



to use:
./log_burner.sh

try "chmod +x log_burner.sh" first if it doesn't work
